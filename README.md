# 👨🏻‍💻 Retail Customer Behavior Analysis

This project represents an end-to-end Data Analytics workflow designed to analyze customer shopping behavior and generate actionable business insights using Python, SQL, PostgreSQL, and Power BI.

The project focuses on customer purchasing patterns, product performance, customer segmentation, subscription behavior, and sales trends to support data-driven business decision-making.

---

# 📌 Project Overview

The objective of this project is to transform raw retail transaction data into meaningful business intelligence by performing:

✅ Data Cleaning & Exploratory Data Analysis using Python  
✅ Business Analysis & SQL Querying using PostgreSQL  
✅ Interactive Dashboard Development using Power BI  
✅ Customer Segmentation & Sales Trend Analysis  
✅ Business Insight Generation & Recommendations

# 🛠️ How to Use This Project

## 1. Clone the Repository

```bash
git clone <your-repository-link>
cd Retail-Customer-Behavior-Analysis
```

---

## 2. Open the Jupyter Notebook

Open:

```text
Customer_Shopping_Behavior_Analysis.ipynb
```

This notebook contains:
- Data Import
- Data Exploration
- Data Cleaning
- Feature Engineering
- Database Connection
- Exploratory Data Analysis

---

## 3. Load Data into PostgreSQL/MySQL

- Create a database in PostgreSQL/MySQL
- Run Python code to load cleaned data into SQL database
- Open:

```text
customer_behavior_sql_queries.sql
```

- Execute SQL queries to answer business questions and generate insights

---

## 4. Connect SQL Database to Power BI

Open:

```text
customer_behavior_dashboard.pbix
```

Create interactive dashboards and visualizations for:
- Revenue Analysis
- Customer Segmentation
- Product Performance
- Sales Trends
- Customer Loyalty Insights

---

## 5. Create Report & Presentation

- Prepare project documentation and business insights report
- Build presentation slides for stakeholder communication
